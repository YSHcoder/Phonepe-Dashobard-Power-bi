#  PhonePe Transaction Insights Dashboard

## 1. Project Title / Headline

 **PhonePe Transaction Insights Dashboard | Power BI**

An interactive Power BI dashboard designed to analyze PhonePe digital payment transactions across India, providing valuable insights into transaction trends, user engagement, and regional payment performance through dynamic visualizations.

---

## 2. Short Description / Purpose

The PhonePe Transaction Insights Dashboard is a comprehensive Power BI project that transforms raw transaction data into meaningful business insights. It enables users to explore transaction volume, transaction value, registered users, and app engagement across different states, districts, years, and quarters.

The dashboard is designed for business analysts, data analysts, and decision-makers who want to monitor digital payment growth, identify high-performing regions, understand customer behavior, and support data-driven business strategies using interactive reports.

---

## 3. Tech Stack

The dashboard was built using the following tools and technologies:

- 📊 **Power BI Desktop** – Main platform used for building interactive dashboards and reports.
- 🔄 **Power Query** – Used for data cleaning, transformation, and preprocessing.
- 🧠 **DAX (Data Analysis Expressions)** – Created calculated measures, KPIs, and dynamic metrics.
- 🗂️ **Data Modeling** – Established relationships between multiple datasets for accurate analysis.
- 📈 **Interactive Visualizations** – Cards, Bar Charts, Line Charts, Maps, Treemaps, Pie Charts, and Slicers.
- 📄 **CSV Dataset** – Source data containing PhonePe transaction and user statistics.

---

## 4. Data Source

### Source:
The dataset used in this project is the **PhonePe Pulse Dataset**, an open-source initiative by PhonePe that provides aggregated and anonymized digital payment statistics across India.

The dataset contains transaction and user analytics across different **States**, **Districts**, **Years**, and **Quarters**, including:

- Digital transaction count
- Transaction amount
- Transaction categories (Recharge & Bill Payments, Peer-to-Peer, Merchant Payments, Financial Services, Insurance, etc.)
- Registered users
- App opens
- Geographic information for state-wise and district-wise analysis

The data was cleaned, transformed, and modeled using **Power Query** before creating relationships and analytical measures in Power BI.

## 5. Features / Highlights

### 📌 Business Problem

India has witnessed rapid growth in digital payments, generating massive amounts of transaction data every day. However, raw data alone makes it difficult to identify regional trends, monitor user engagement, and evaluate the performance of digital payment services across different states and districts.

---

### 🎯 Goal of the Dashboard

The primary objective of this dashboard is to provide an interactive and data-driven view of PhonePe's digital payment ecosystem by:

- Tracking transaction volume and value.
- Monitoring registered users and app engagement.
- Comparing payment activity across states and districts.
- Identifying yearly and quarterly growth trends.
- Supporting business decisions with meaningful visual insights.

---

### 📊 Walkthrough of Key Visuals

#### 💳 KPI Cards
Displays total transaction amount, total transaction count, registered users, and app opens to provide a quick overview of business performance.

#### 🗺️ State-wise Transaction Analysis
Interactive map and charts showing transaction value and transaction count across Indian states.

#### 📍 District-wise Performance
Highlights the top-performing districts based on transaction activity and user engagement.

#### 📈 Quarterly & Yearly Trends
Line and bar charts showing the growth of digital transactions over different years and quarters.

#### 🥧 Transaction Category Analysis
Breaks down transactions by payment categories, helping identify the most frequently used services.

#### 🎛️ Interactive Filters
Users can filter the dashboard by **Year**, **Quarter**, **State**, and **Transaction Type** for detailed analysis.

---

### 💡 Business Impact & Insights

- Identifies high-performing states and districts based on transaction volume.
- Tracks the growth of digital payments across different time periods.
- Monitors customer engagement using registered users and app opens.
- Helps businesses understand regional payment behavior and customer adoption.
- Supports strategic planning through interactive business intelligence and data visualization.
